# Lamborghini Sales Dashboard

## Project Description
This Power BI dashboard provides a comprehensive analysis of Lamborghini sales data from 2020 to 2025. Designed for automotive industry stakeholders, this visualization showcases key performance indicators (KPIs), sales trends, and regional performance metrics for Lamborghini vehicles. The dashboard aims to offer insights into customer preferences, market dynamics, and sales forecasts.

### Key Features:
- **Interactive Charts**: Users can interact with dynamic charts that display sales performance over different periods.
- **Data Filters**: Apply filters based on region, model, and date range to analyze specific segments of the data.
- **Forecasting**: Utilize advanced analytics algorithms to project future sales.
- **User-Friendly Interface**: Intuitive design that enhances user experience and facilitates data exploration.

### Technologies Used:
- **Power BI**: Primary tool for data visualization and dashboard creation.
- **DAX**: Data Analysis Expressions used for calculations and data modeling.
- **Azure**: Cloud services utilized for hosting and data storage.

This project is ideal for stakeholders looking to understand trends in the luxury car market and make informed business decisions based on data insights.
